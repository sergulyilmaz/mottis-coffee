## Proje Amaci ve Aciklamasi

**Mottis Coffee**, kahve dukkanlari icin gelistirilmis kapsamli bir sadakat (loyalty) ve yonetim sistemidir. Sistem dort ana bilesenden olusmaktadir:

- **Musteri Mobil Uygulamasi (Flutter)** - QR kod ile pul toplama, menu ve magaza goruntuleme
- **Barista Mobil Uygulamasi (Flutter)** - QR tarama ile musteri pullarini yonetme
- **Admin Web Paneli (React)** - Magaza, personel, urun ve kampanya yonetimi
- **Backend API (Node.js/Express)** - Merkezi sunucu ve veritabani yonetimi

**Sadakat Mantigi:** Musteri her kahve aldiginda barista QR kodunu tarar, 1 pul eklenir. 8 pul = 1 ucretsiz ikram.

## Kurulum Adimlari

### On Kosullar
- Node.js (v18+), PostgreSQL (v14+), Flutter SDK (v3.29+), Git

### Kurulum
```bash
# 1. Repoyu klonlama
git clone https://github.com/KULLANICI_ADI/mottis-coffee.git

# 2. Veritabani kurulumu
createdb mottis-db
psql -U postgres -d mottis-db -f mottis-backend/schema.sql

# 3. Backend
cd mottis-backend && npm install
cp .env.example .env   # .env degerlerini duzenleyin
npm run dev             # Port 3000

# 4. Admin Paneli
cd mottis-admin && npm install
npm run dev             # Port 5173

# 5. Flutter Uygulamalari
cd mottis_flutter && flutter pub get && flutter run
cd mottis-barista && flutter pub get && flutter run
```
